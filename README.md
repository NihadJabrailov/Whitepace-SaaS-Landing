A clean, professional landing page for a modern SaaS product — designed to convert visitors into customers.

✅ Fully Responsive — optimized for all screen sizes (mobile, tablet, desktop)
✅ Pixel-Perfect Design — faithful implementation of the Whitepace Figma design
✅ Reusable Components — clean, modular React component architecture
✅ Smooth Animations — subtle transitions and scroll effects for polished UX
✅ Mobile-Friendly Navigation — hamburger menu with smooth open/close behavior
✅ Type-Safe Codebase — full TypeScript support throughout
✅ Fast Build — powered by Vite for lightning-fast HMR and production builds

bashgit clone https://github.com/NihadJabrailov/Whitepace-SaaS-Landing.git

3. Start the development server
